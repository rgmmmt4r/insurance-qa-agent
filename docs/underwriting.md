# 核保規則摘要（虛構範例，供內部參考）

> 本文件為虛構範例資料，僅供 AI Agent 展示用途，與任何真實保險公司核保政策無關。

## 核保基本原則
核保是評估投保風險、決定是否承保及承保條件的程序。本公司依被保險人的年齡、健康狀況、職業、投保金額等因素綜合評估。

## 免體檢額度
- 30 歲（含）以下：累計壽險保額 800 萬元以下免體檢
- 31～50 歲：累計壽險保額 500 萬元以下免體檢
- 51～70 歲：累計壽險保額 300 萬元以下免體檢
- 超過上述額度者，須依規定接受體檢與相關檢查

## 職業等級分類（影響意外相關保障）
- 第 1～2 級：內勤、文書、教師等，標準承保
- 第 3～4 級：技術員、外勤業務等，加費承保
- 第 5～6 級：高空作業、礦工等高風險職業，個案評估或拒保
- 拒保職業：依公司公告之高度危險職業表

## 健康告知與加費
- 投保時須據實填寫健康告知書
- 體位（BMI）、血壓、血糖等指標異常者，可能採加費承保或批註除外
- 重大病史（如癌症、心臟病史）者，依個案評估，可能延期、加費或拒保

## 核保決定類型
1. **標準承保**：依標準費率正常承保
2. **加費承保**：因風險較高，加收額外保費
3. **批註除外**：特定部位或疾病不予保障，其餘正常承保
4. **延期**：暫緩承保，待狀況穩定後再評估
5. **拒保**：風險過高，不予承保

## 與其他文件之關聯
- 各商品投保年齡與保額上限，請參閱對應商品條款摘要。
- 等待期與除外責任，與核保決定併同適用。
